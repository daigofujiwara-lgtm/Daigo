# 議事録作成スキル

以下の手順で会議の議事録を整理・作成してください。

## 入力

ユーザーが提供するもの（いずれか）:
- 会議中のメモ・走り書き
- 録音テキスト・文字起こし
- 箇条書きの断片的なメモ

以下のフォーマットで整理した議事録を作成してください：

## 実行手順

1. ユーザーからメモや文字起こしを受け取る
2. 上記フォーマットに整理する
3. 整理した議事録を表示する
4. ユーザーに「ファイルに保存しますか？」と確認する
5. 保存する場合は `/Users/fujiwaradaigo/Desktop/クロードCode/議事録/YYYYMMDD_会議名.md` に保存する
6. 必要に応じてSlackの該当チャンネルに共有するか確認する
