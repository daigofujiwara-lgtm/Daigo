# 提案資料構成スキル

新規クライアント向けのインフルエンサー施策提案資料の構成を自動生成します。

## 入力

ユーザーに以下を確認してください：
1. クライアントの業種・商材（例：美容、食品、アパレルなど）
2. 目的（例：新規獲得、認知拡大、リピート促進）
3. 予算感（例：成果報酬型 / 予算型）
4. ターゲット（例：20代女性、30代主婦など）

情報がない場合はナハトの標準的なInstagram施策として構成を作成してください。

## 出力フォーマット

---

# 【クライアント名 or 業種】Instagram インフルエンサー施策 提案資料構成

## 1. エグゼクティブサマリー（1枚）
- 課題・提案概要・期待成果

## 2. 現状課題の整理（2枚）
- クライアントの課題
- 市場トレンド・競合状況

## 3. ナハトの強み・実績（2枚）
- 登録インフルエンサー15,000人・合計13億フォロワー
- 年間2,100件以上の実績
- 成果報酬型モデル・インフルエンサーアフィリエイト国内シェア約80%
- 業種に近い成功事例を記載

## 4. キーパーソンマーケティング説明（2枚）
- 3軸選定基準（基本条項マッチ度・商品マッチ度・エンゲージメント）
- 従来施策との違い

## 5. 施策提案（3枚）
- ターゲットインフルエンサーの選定方針
- 投稿形式（リール中心）
- コンテンツ方向性

## 6. 実施スケジュール（1枚）
- キャスティング → 制作 → 投稿 → 効果測定（約4〜6週間）

## 7. KPI・効果測定（1枚）
- リーチ / エンゲージメント率 / CV数 / CPA

## 8. 費用・プラン（1〜2枚）
- 成果報酬型 / 予算型の2プラン

## 9. 次のステップ（1枚）
- ヒアリング → 提案書精緻化 → 契約 → 実施

---

## 実行手順

1. クライアント情報をユーザーに確認する
2. 上記フォーマットに情報を埋めて構成を出力する
3. 「各スライドの詳細テキストも作成しますか？」と確認する
4. 必要に応じてMarkdownファイルとして保存する
