# インフルエンサーキャスティングスキル

案件に最適なインフルエンサーの選定条件を整理し、キャスティング要件書を作成します。

## 入力

ユーザーに以下を確認してください：
1. 商材・サービス名
2. ターゲット層（年齢・性別・興味関心）
3. プラットフォーム（Instagram / YouTube / TikTok / X）
4. 施策タイプ（成果報酬型 / 予算型）
5. 予算感・希望投稿数

## 実行手順

1. 情報をヒアリングする
2. ナハトのキーパーソンマーケティング3軸で整理する
3. キャスティング要件書を出力する
4. Markdownファイルとして保存するか確認する

## 出力フォーマット

---

# キャスティング要件書

**商材**:
**作成日**: YYYY年MM月DD日
**担当**: 藤原汰瑚
