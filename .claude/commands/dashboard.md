# ダッシュボード生成スキル

ExcelやCSVファイルを読み込んで、HTMLダッシュボードを自動生成します。

## 入力

ユーザーに以下を確認してください：
1. ファイルパス（Excel / CSV）
2. ダッシュボードの目的（例：売上管理、インフルエンサー管理、案件進捗など）
3. デザインの雰囲気（例：シンプル、派手・ランキング形式、カンバンなど）
4. 表示したい主要指標（例：売上、CPA、フォロワー数など）

## 実行手順

1. ファイルを読み込んでデータ構造を確認する
2. ユーザーの要件に合わせてHTMLダッシュボードを生成する
3. `/Users/fujiwaradaigo/Desktop/クロードCode/アプリ開発/` に保存する
4. ブラウザで開いて確認する

## 実装する機能（標準）

- フィルタリング機能（媒体、事業部、担当者など）
- ソート機能（数値列）
- 検索機能
- レスポンシブデザイン
- Chart.jsを使ったグラフ表示（必要に応じて）

## デザインパターン

- **シンプル**: 白背景、テーブル中心、青系カラー
- **ランキング**: 金銀銅メダル、グラデーション、順位バッジ
- **カンバン**: 事業部・カテゴリ別カラム、カード形式
- **ダーク**: 黒背景、蛍光色アクセント、モダンUI
